# Project Blueprint: MONEYBACK - 연말정산 행동 추천기

## Overview
This project is a web-based "Year-End Tax Adjustment Action Recommender" designed to help users maximize their tax refunds. It provides an intuitive interface to input personal income and expenditure details and suggests optimal action plans based on these inputs.

## Current Application State
The application is a single-page web application built with HTML, Tailwind CSS, and JavaScript. It currently features:

*   **Responsive Design:** Utilizes Tailwind CSS for a modern, responsive layout that adapts to different screen sizes.
*   **Themed UI:** Implements a light theme with a specific color palette defined in `tailwind.config`.
*   **Input Forms:** Several sections for user input:
    *   **Income Information:** Annual income input.
    *   **Family & Dependent Information:** Checkbox for spouse deduction, and a counter for minor children.
    *   **Expenditure Details:** Inputs for credit card usage, debit card/cash receipts, traditional market usage, and public transportation usage.
    *   **Financial/Savings Products:** Inputs for pension savings/IRP contributions and housing subscription savings.
*   **Call to Action:** A prominent button to "Get Refund Action Plan".
*   **Informational Banner:** A guide banner advising users to use 'Year-End Tax Adjustment 미리보기' for accurate results.
*   **Footer:** Editorial information and links to terms of service, privacy policy, and customer support.
*   **Mobile Navigation:** A fixed bottom navigation bar for mobile views with icons for Home, Calculation, Tax Guide, and My Info.
*   **Assets:** Placeholder image assets for data-alt requirements.
